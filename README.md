# 🎯 Quiz Application (Java Console-Based)

## 📌 Overview

This is a **console-based Quiz Application** developed using **Java**.
The application allows users to attempt multiple-choice questions with a time limit and evaluates their performance instantly. It also maintains a **leaderboard system** to track and display top scores.

---

## ✨ Features

* 🧑 User name input
* 📋 Menu-driven interface
* ❓ Multiple-choice questions (MCQ)
* ⏱️ Timer-based questions (auto skip after 20 sec)
* ✅ Instant answer validation
* 📊 Score calculation
* 🏆 Leaderboard (sorted – highest score first)
* 💾 File handling (stores scores in `scores.txt`)

---
